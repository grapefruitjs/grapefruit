Plays a loaded audio clip

Parameters:

  • id String | Object

    The id of the sound clip to play. You can also pass the object returned from a previous play

  • options Object

    The options object you can pass properties like "loop," "volume," "channel"

  • callback Function

    The callback to call after the sound finishes playing

Returns:

Object: The object returned can be passed to any audio function in the first parameter to control that audio clip
