_complete

() private

Inherited from fx.camera.Effect: src\fx\camera\Effect.js:71

Called when the effect finishes to call the registered callback (if there is one). If the callback explicitly returns false then .stop() will not be called. done will still be set to truebut the effect will not be removed from the display.

This is useful if you want to run an animation and keep the final state active until you manually remove the item with .stop(). For example: fading to black then running some async process, then removing the black manually.
